# Insurance Management System

## 📌 Description
Web-based insurance management system developed using PHP and MySQL.

## 🛠 Technologies Used
- PHP
- MySQL
- HTML/CSS
- XAMPP

## 🚀 How to Run
1. Install XAMPP
2. Put project folder inside htdocs
3. Import database
4. Run http://54.251.71.21/insurance-app
